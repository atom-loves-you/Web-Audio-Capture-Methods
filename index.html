There are lots of methods you theoretically can capture audio using web standards. Each of these methods is an attempt mainly at trying to get iOS Safari to do... pretty much anything. As of 4/13/17, mobile Safari seems to only work by recording a video via the camera app and using that to play back in the audio element. WebRTC is currently unsupported by any version of Safari, but works fine across the other browsers I've tested.
